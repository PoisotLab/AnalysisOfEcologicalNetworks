BIO3043 / BIO6037

Practical Analysis of Ecological Networks

This book offers an in-depth introduction to the analysis of ecological networks. By using actual datasets, it covers the core principles of network analysis, and some advanced applications related to network prediction and spatial analysis. This book is built around code and data, to offer immediately applicable insights into how ecological networks should be analyzed.
